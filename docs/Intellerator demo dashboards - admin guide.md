## User access administration

Viewing and Adding users  

1. Get email addresses from the partner/customer
2. Log into Power BI Web \- Codis B2B \- [https://app.powerbi.com/home?ctid\=b45363d0\-8aae\-43ca\-9e1f\-ddb6097861ce](https://app.powerbi.com/home?ctid=b45363d0-8aae-43ca-9e1f-ddb6097861ce)
3. Log in using “Microsoft 365 \- Power BI Install“ credentials from 1Password
4. Select Workspaces icon on left hand side, then search for the demo (e.g. BP name)
5. View users: Update App \> Audience \> Edit Audience \- see list of users
6. Add users: Update App \> Audience \> Edit Audience \> add users \> click Update app \- make a note of the URL generated.
7. Re\-add users: type name of user \> click Update app
8. From left hand menu select workspace of that BP
9. Set row level security (RLS): For each semantic model (Financial Statements, Nominal Transactions, SOP): click on 3 dots \> Security \> type in name \> select user(s) \> Save
Note: users must have a Power BI Pro licence.  
To get the URL, Go to Update App \> scroll down within Setup \> Copy link.  
## Emailing new users

Once users are added, use this template to email them:  

\-\-\-  
Hi \{name\},  
Here’s the link to access Intellerator for customer demos, whether you're presenting remotely or on site: \{URL\}.   
What to keep in mind:- You'll need a a Power BI Pro licence to demo Intellerator. Many organisations already have this as part of their Microsoft 365 subscription, so it's worth checking. Here's how:
- Sign in to [https://powerbi.microsoft.com](https://powerbi.microsoft.com/).
- Click the gear button in the top ribbon.
- Click "manage personal storage".
- From this window, it should say your name and "Pro User".

- Power BI can be accessed via any web browser.
- Your user access has been set up, but Microsoft may take up to 24 hours to activate it. You may receive an email with the app link once it's ready.

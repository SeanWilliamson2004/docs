To install and configure customised Excelerator for MJ french, please follow the below.  

- Please download and Extract the zip file on user machine.
- Install Excelerator 3\.2\.175 or use the files in the zip document to register the addin.
- Once the addin is registered, Copy customisation folder to C drive(or some other location accessible by the user profile)
- Open Excel \> Excelerator \> display settings button \> Plugins and enter the path of the customisation folder.
- This will allow Excelerator to read those dll files.
- Make sure that Sage 200 Administration has the attached sdbx installed from the zip document.

The software would be ready for the user to test.[Codis Customisation.zip](https://codislimited.sharepoint.com/sites/Wiki/Support/Backup/Rhys%20Bhan/Codis%20Customisation.zip)

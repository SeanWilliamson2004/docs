PPS Offline Activation

1 Rename the licence file to licenceold.xml  

2 open system settings click register now, cick advance and copy registration Code.  

3 open company in CRM, go to licences\-\>enterprise\-\>delete old Server with the same name.  

4 click on add server from top right sidde on crm  

5 enter the server name and add the offline registration code in computer ID and click save  

6 after a new server is added click add offline server.  

7 another new server will be added and generate offline activation on that new server.8 delete the server you added in step (4\)

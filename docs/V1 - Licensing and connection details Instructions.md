# Licensing

## Open Excel as Administrator (see below)

![](images/Support_Support_Wiki_Images_Images_for_solutions_v1_1.jpg) 

## Click on Administration tab in NL Journals via Add\-Ins tab in Excel (see below)

![](images/Support_Support_Wiki_Images_Images_for_solutions_v1_2.jpg) ## Select License on File tab.

![](images/Support_Support_Wiki_Images_Images_for_solutions_v1_3.jpg) ## Then select License Method and please make sure if its PC select Method A and for Laptop Method D.

![](images/Support_Support_Wiki_Images_Images_for_solutions_v1_4.jpg) ## Activation Code (at this stage call our licensing team on 0208 8610 610 option2 and provide them the code and they will guide you from there)

# 

![](images/Support_Support_Wiki_Images_Images_for_solutions_v1_6.jpg)  # Entering connection detail (see below)

![](images/Support_Support_Wiki_Images_Images_for_solutions_v1_7.jpg) ## After entering details Click on Test (to check if details entered are correct).

# 

## Once connection Successful, click OK, close excel and open Excel as normal.

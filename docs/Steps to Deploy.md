**✅** **Steps to Deploy a Build**

  


**🔹** **Step 1: Open Deployment Portal (Microsoft Azure Storage Explorer)**

Navigate to the deployment portal from your browser or application launcher.

*Click on the deployment dashboard or app to begin.*

\-Visual Studio Enterprise \-Codis Main ([administrator@codis.co.uk](mailto:administrator@codis.co.uk))

\>Storage Accounts

                \>Codis

                                \>Blob Containers

                                                \>$web  


**🔹** **Step 2: Select the Deployment Section**

Navigate to Excelerators/Sage200/

*Look for previous build (StandardSage200vSPE2019ExceleratorSuite.msi)*

  


**🔹** **Step 3: Choose the Build to Deploy**

Select the appropriate build/version you want to deploy.

*Confirm the build ID, timestamp, or changelog before proceeding.*

  


**🔹** **Step 4: Configure Deployment Settings**

Delete and replace with the new version

*Make sure to double\-check target settings to avoid mis\-deployments.*

  


**🔹** **Step 5: Initiate Deployment**

*Monitor logs or status to confirm the deployment was successful.*

**📝** **Notes**

- Deployment usually takes 2–5 minutes.
- Always download the uploaded version to check the version.

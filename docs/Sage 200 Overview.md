**This Wiki page contains all useful information related to Sage 200\.**

## [**Sage 200 Product Range**](https://gb-kb.sage.com/portal/app/portlets/results/viewsolution.jsp?solutionid=200427112404508&hypermediatext=null)

### 

### Sage MMS

This product preceded Sage 200 

### Sage 200 Extra On\-Premise

This is the original Sage 200\. It requires a server on the customers site to host Sage 200\. Software is installed on the user's PC and the server. It allows customisation. 

### Sage 200 Extra On\-Line

This offers the same range of functionality as Sage 200 Extra On\-Premise, but no server is required. Sage hosts the server side. It allows customisation. This software is leased. 

### Sage 200c Standard

This version of Sage 200 offers cut\-down functionality and the server side is hosted by Sage. Customisation is not allowed. This software is leased and offers a low\-cost path for Sage 50 users to upgrade.

### Sage 200c Professional

Sage 200c Professional is a highly flexible solution that is ideal for SME’s ranging from 10 – 200 staff members with approximately £1 million – £50 million turnover. It covers all business sectors, enabling the solution to grow alongside your company. For businesses that are cross\-departmental or have notably complex processes, Sage 200c Professional is the ideal Solution. Sage 200c Professional has also been designed to work with a range of different software applications, enabling businesses with specific industry needs to add modules to cover in\-depth areas such as project accounting, commercials, construction, manufacturing, wholesale and retail.

# 1\. Open an opportunity on CRM.

## 1\.1 New Opportunity.

**\- Product Family** (Select which product they are purchasing).

**\- Description**

**\-** Type in the **details**. (This may include which type of licenses they require)

**\- \#Licenses:** How many licenses do the want? (If it is for a webinar, put 0 licenses)

**\-** **Days:** Usually you would put in 0 consultancy days. (Unless stated otherwise)

**\- Priority:** You would select 'Normal' unless stated otherwise.  

**\- Assign** to yourself or relevant person.

**\- Team:** Select Sales

**\- Close By:** Select a day that is in the end of the next month.

Finally, press **save**.

# 2\.New order

## 2\.1 The Order Form (Standard or Terminal/Citrix)

 ![Purchase Order- Standard.TC Screenshot.PNG](images/Sales_Sales_Wiki_Images_Order_Processing_on_CRM_Purchase_Order-_Standard.TC_Screenshot.PNG)

 Before you process the order, look at your order form and look at the box that says 'RDT Server Install' as shown in the screenshot above. 

If it says **No** then it is a standard license. If it says **Yes** then they are looking for a Terminal/Citrix license. 

## 2\.2 Order

![Order Screenshot.PNG](images/Sales_Sales_Wiki_Images_Order_Processing_on_CRM_Order_Screenshot.PNG) 

Go to the green tab on top and click on 'order' then click on 'new' located on the side.

Fill in **description**, you type in which modules they want.

**Delivery date:** This is usually the same day you are receiving the order.

**PO Reference:** Type in the puchase order reference which can be found on the actual purchase order. Other option is to type SO (Sales Order) then put the date next to it. **(i.e. S0151117\)**        If you received the order from Sage, then you can use the Tracking number (long digit code) as the PO Reference.          

Then click **next**.

Fill in the boxes required and press save \& new, or just save if there is one order.![New Line.PNG](images/Sales_Sales_Wiki_Images_Order_Processing_on_CRM_New_Line.PNG)

 Next you will click on 'new line item'.

In the bottom part of the screen called Line Items, you will enter the 'Product Name', 'quantity', and double check if everything if fine. Press save, or save and new if you need to add more products.

## 2\.3 Licenses

### **2\.3\.1 Standard License**

Now you must go to 'License' tab at the top.

The new modules will be highlighted in yellow and located above the other modules.

Click 'Renew' on the right hand side of your screen. 

Click on the new module and then enter the 'expiry date'. In the 'Note' box type in 'new order'.

Press 'save', then go to 'edit summary', as shown the screenshot above, click 'Live', and then press save on the right hand side of your screen.

 ![License 'Live' Screenshot.PNG](images/Sales_Sales_Wiki_Images_Order_Processing_on_CRM_License_Live_Screenshot.PNG)

### 2\.3\.2 Enterprise License

If the order is for an enterprise license then you will have to go onto licenses and select the enterprise tab on the side.

Click on the module and then select renewal on the right hand side. The expiration date will be automatically selected. Do not change the date.

In the order box, type in New Order and then the Purchase Order number. Press save and the license should be activated.

So now the licenses will be available to activate once the user calls in. 

# Final Bits

### 2\.4\.1 Close the opportunity

File an email copy of the purchase order and save onto the documents tab.

Go back to the opportunities tab and select reassign on the right hand side. 

![Reassign Screenshot.PNG](images/Sales_Sales_Wiki_Images_Order_Processing_on_CRM_Reassign_Screenshot.PNG) 

In the image above, there are two options to finish off. Click on stage and select 'Sales Closed', and then click on status and select 'Won'. Then press save. 

Ensure the opportunity is closed. 

*(If the sales has not been closed then in the* *Status* *tab you will select '**Lost**')*

Go back to opportunities and change the Type to from 'prospect' to 'customer'.

The alignment date should be updated to match the Sage renewal date.

### 2\.4\.2 Email to Reseller

Now you must send an email to the reseller and making them aware that you have processed the order. 

So go to the resellers account,

Go to the people tab.

Select a person (preferably an account manager).

Select the communication tab then, new email on the right hand side.

![Email to Reseller Screen shot.PNG](images/Sales_Sales_Wiki_Images_Order_Processing_on_CRM_Email_to_Reseller_Screen_shot.PNG) 

In the email, the subject line should include the **Order number or the name of the customer**which can be found on the actual Purchase Order.

Select the template, if the order is for Excelerator for Sage 200, then select the 'Order Confirmation 'AAA'. 

Then select 'Send using Outlook'. 

Make sure you 'CC' bpsales **(If it is a Sage 1000 order DO NOT copy bp sales).**

Then sign off with 'kind regards', and press send.

 ![Sending email screenshot.PNG](images/Sales_Sales_Wiki_Images_Order_Processing_on_CRM_Sending_email_screenshot.PNG)

**After you have sent the email. Go to the Sage 200 account on CRM and copy and paste a copy of the order onto the** **documents tab** **page.**

### 2\.4\.3 Updating the Company Summary Page

**![Company Summary screenshot.PNG](images/Sales_Sales_Wiki_Images_Order_Processing_on_CRM_Company_Summary_screenshot.PNG)** 

Finally, go to the company's summary page and update the following (as shown in the screenshot):

**Sage Renewal Date:** This can be found on the order form.

**Alignment Day:** The day of the renewal date.

**Alignment Month:** The month of the renewal date.

**Use Alignment Date:** Tick the box.

# 3\. Creating a consultancy case on CRM.

## 3\.1 Part 1

\- Most cases are assigned to Ali.

In the description box, type in how many days consultancy. In the Sales Order example on the right is shows in the 'product code', 2\.0 consultancy, 2\.0 consultancy and 1\.00 consultancy. This means there are 5 days consultation service, so you type in '5 days consultancy' in the box as shown in the image in '2\.2 Part 2'.

![](images/Sales_Sales_Wiki_Images_Order_Processing_on_CRM_Sales_Order_Screenshot.PNG) 

## 3\.2 Part 2

Go to Progress Case/edit case tab on the right hand side.

Enter the Customer PO: This is the 'PO Number' on the sale order. 

Enter the Codis sales Order: This is the 'order number' on the sales order.  

Next, go to the Documents tab at the top, and get a email copy of the Sales order and add a copy into the Documents page.

Finally, take down the Ref ID, note it down onto the Sales Order, and then email the sales order along with the Ref ID in the

email to the **Support Team** and **CC Ali** as well.

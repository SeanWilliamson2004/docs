A default Excelerator template is a spreadsheet template for Excelerator that comes installed with the software. A **release** template is one that is installed in the users pc, a **demo** template is one we use for web demos. 

## Location

You can find the current release Excelerator templates as follows. 



| **Name** | **Description** | **Location** |
| --- | --- | --- |
| Sage 200 V3 | Templates that will be with the Excelerator that comes packaged with Sage 200 V3 | [Sage 200 V3 Latest Templates](file://///cd01sr16104/Excelerator_Templates_Latest/V3%20Sage%20200%20Excelerator%20Templates) |
| Sage 1000 V3 | Templates that will be with the Excelerator that comes packaged with Sage 1000 V3 | [Sage 1000 V3 Latest Templates](file://///cd01sr16104/Excelerator_Templates_Latest/V3%20Sage%20500-1000%20Excelerator%20Templates%20work%20for%20Standard%20and%20Enterprise) |

## Guidelines

These are guidelines for to be used when creating Excelerator templates. The same guidelines should be used for Sage 200 Excelerator and V3 Sage 500/1000 Standard and Enterprise Excelerators. 

### Demo Template

These are templates used for demos. Ubuntu can be used because we have it installed on our computers. 



| **Area** | **Guidelines** |
| --- | --- |
| Font | Single Ubuntu Light, Multiple Ubuntu |
| Size | 11 |
| Logo | Top right \- white surround and background \- right border aligns with grid |
| Gridlines | Unticked |
| Field Labels | In header aligned to left |
| Headings | Headings in columns centre aligned \- bold |
| Tooltips | Off |
| Data entry fields | no fill \- font Ubuntu |
| Copyright message | Using this template constitutes the acceptance of the licence terms and conditions. Copyright © 2014 Codis Ltd. All rights reserved. |
| Vertical borders | Defined in the grids |
| Colours | Default 217, 232, 247 |
| Grid colours | Enable range colour \& enable alternate column |

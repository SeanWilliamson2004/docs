Currently there are 3 licence types: **Standard**, **Enterprise** and **Terminal/Citrix**. This process will be looking through Codis Licensing Teams point of view. 

## Standard

The software is installed locally on the user's PC or laptop. One standard licence covers one user on one pc, if a user wants to use the software on multiple PCs or laptops, they will need to purchase multiple standard licences. 

### Activating Standard License

#### V1

1. Confirm Name and Module for activation
2. Confirm the Computer ID
3. Click Activate by Phone
4. Type in Serial Code
5. Give out 2 Activation Codes

For further information click on the [Licensing Manual for V1](https://codislimited.sharepoint.com/sites/Wiki/Licensing/Licensing%20Wiki/Documents/Codis%20CRM%20Licensing%20Manual%20v1.1.pdf). 

#### V3

1. Click on the computers
2. Click on the User Name
3. Click on Request
4. Confirm the details; Name, Computer ID, Module \& License ID
5. Click Net Activation
6. First Save and then Copy
7. Paste into notepad and send in email

For further information click on the [Licensing Manual for V3.](https://codislimited.sharepoint.com/sites/Wiki/Licensing/Licensing%20Wiki/Documents/Codis%20Excelerator%20V3%20Licensing%20Instructions.pdf) 

### Demo

A demo licence is a type of standard licence that can be issued to a customer without making an order, for a limited time. It restricts the features of the software, so that the customer will need to purchase a standard licence to use all of the features.

## Enterprise

Codis' server\-based solution. The software is installed on multiple Users PCs, and the licensing is controlled on a server at the **company's site**. Licences are controlled by accessing the Integrated Platform. The administrator can select which users are allowed to use which modules. 

### Activating Enterprise License

This is a two part process.

1. Ask for the Server Name
2. Confirm the Computer ID (6 digit number)
3. Type Module Name
4. Type Serial Code give 2 activation codes, (do this twice)
5. Repeat steps 2\-4 until all the modules they have requested are activated
6. Update IP User list; type Serial Code and give 1 activation code.

## Terminal/Citrix

The software is installed once on a server. Other users (*clients*) then access the software via [Remote Desktop Services](https://technet.microsoft.com/en-us/windowsserver/ee236407.aspx) or [Citrix XenApp](https://www.citrix.com/products/xenapp/overview.html). 

Sometimes there are many clients, and all of them accessing one server will slow down the server. In this case, the software is installed on multiple servers (a [server farm](https://en.wikipedia.org/wiki/Server_farm)), each a duplicate of the first. When a client then needs to use Codis software it is automatically connected to the server with the most free resources.

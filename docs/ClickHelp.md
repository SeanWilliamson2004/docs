**Click Help**

For each project, should follow these instructions.

1\. Current projects \> Settings from drop down \> Topic master page \> Check use parent level settings.  

![Parent level.png](images/PublishingImages_Pages_ClickHelp_Parent_level.png)  

2\. Tools \> Portal settings \> Branding \> Reader interface \> Choose the project from drop down \> tick use portal settings (right of screen) for all the projects and publications.  

![PR level.png](images/PublishingImages_Pages_ClickHelp_PR_level.png)  

![Portal.png](images/PublishingImages_Pages_ClickHelp_Portal.png)  

3\. Select the project \> Edit mode \> Select the whole page text \> Use clear formatting for each page on the project to apply CSS.  

![Formatting.png](images/PublishingImages_Pages_ClickHelp_Formatting.png)

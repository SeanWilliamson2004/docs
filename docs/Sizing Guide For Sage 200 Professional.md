**Sizing Guide For Sage 200 Professional**

This is a pre\-sales resource

Sage have provided the following guide that explains the recommended Sage 50 data sizing to help assess migration to the Sage 200 platform.  

This can also be used for sizing Sage 200 installations too.

[Sizing Guide for Sage 50 Accounts to Sage 200 Migration](https://gb-kb.sage.com/portal/app/portlets/results/viewsolution.jsp?solutionid=231129145642017&page=1&position=0&q=sizing%20guide)

**Sample sizing estimate (sent to Gelato Italia Wholesales Limited and Lavender Medical Ltd)**  

**Windows Specifications**  
Edition: Windows Server 2022 Standard  
Version: 21H2  
OS Build: 20348\.2402  
Windows: 64bit (Pro and Enterprise editions)  
SQL Server: Microsoft SQL Server 2022 / 2019 (Standard or Enterprise Edition)

**Processor:**AMD EPYC 7543P 32\-Core Processor 2\.80 GHz  
RAM 32\.0 GB  
System type: 64\-bit operating system, x64\-based processor

**Disk Space**  
Drive C: 200GB  
Drive D: 500G  

The above estimate should be **tailored** for each site and sent with the latest Sage 200 Professional System Requirements specification.

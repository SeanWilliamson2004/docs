Developers documentation will frequently require code to be included and this code should appear correctly formatted with context colouring.

To embed VB.NET code in SharePoint wiki pages the following steps should be followed:

1. In Visual Studio, copy the code into the clip board.
2. Goto [http://hilite.me/](http://hilite.me/)
3. Paste the VB.NET code into **Source Code** window
4. Select Language "VB.NET" and Style "Colourful"
5. Click Highlight
6. Either \- to display the code in a snippet box:
1. Copy everything from the HTML window
2. In Wiki, select Embed Code and paste into the window

8. Or, to show the code but not in a snippet box:
1. Select the code in the **Preview** and copy
2. Paste directly into the wiki page.

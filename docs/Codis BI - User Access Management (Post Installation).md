# 


> - **This page is about managing access of \*Codis\* users to live customer dashboards, and not for managing end user access.**
> - ****There are three places where the User Access to Customer Dashboards should be managed on powerbi.com:****

1. **Customer Workspace**
2. **Customer Data Gateway**
3. **Customer Data Sources**

1. #### **Granting user Access for Customer Dashboards**
1. **Customer Workspace**
1. Go to powerbi.com.  
 You can use this URL: [https://app.powerbi.com/home](https://app.powerbi.com/home)
2. Make Sure it is Signed in by your Codis Email ID.  
 You will see a Screen like this:  
 ![1.1.2 Home Page](images/Support_Support_Wiki_PowerBI_Images_Codis_BI_-_User_Access_Management_Post_Installation_1.1.2_Home_.png)
3. Click on Workspaces:  
 ![1.1.3 Workspaces](images/Support_Support_Wiki_PowerBI_Images_Codis_BI_-_User_Access_Management_Post_Installation_1.1.3_Open_workspaces.png)
4. Select Company workspace:  
 ![](images/Support_Support_Wiki_PowerBI_Images_Codis_BI_-_User_Access_Management_Post_Installation_1.1.4_Select_company_workspace.png)
5. Click on Access:  
 ![](images/Support_Support_Wiki_PowerBI_Images_Codis_BI_-_User_Access_Management_Post_Installation_1.1.5_Access.png)
6. Type Email Address of the User to which the access is to be Granted. And Select Admin from the Dropdown.  
 ![](images/Support_Support_Wiki_PowerBI_Images_Codis_BI_-_User_Access_Management_Post_Installation_1.1.6_Adding_User.png)
7. Click on "Add" and then Close the window from bottom left "Close" Option.

3. **Customer Data Gateway**
1. Click on Settings.  
![](images/Support_Support_Wiki_PowerBI_Images_Codis_BI_-_User_Access_Management_Post_Installation_1.2.1_Settings.png)
2. Click on Manage Gateway.
3. Click on On\-Premises Data Gateway  
![](images/Support_Support_Wiki_PowerBI_Images_Codis_BI_-_User_Access_Management_Post_Installation_1.2.3_On-Premises_data_gateway.png)
4. Then Click on 3 Dots next to the customer gateway name. Then Click on Manage Users.  
![](images/Support_Support_Wiki_PowerBI_Images_Codis_BI_-_User_Access_Management_Post_Installation_1.2.4_Manage_Users.png)
5. Add User here.  
![](images/Support_Support_Wiki_PowerBI_Images_Codis_BI_-_User_Access_Management_Post_Installation_1.2.5_Add_User.png)
6. Then make the user Admin.  
![](images/Support_Support_Wiki_PowerBI_Images_Codis_BI_-_User_Access_Management_Post_Installation_1.2.6_Make_Admin.png)
7. Then Click on "Share".

5. **Customer Data Sources**
1. Click on Data Sources.  
![](images/Support_Support_Wiki_PowerBI_Images_Codis_BI_-_User_Access_Management_Post_Installation_1.3.1_Data_Sources.png)
2. Click on 3 Dots next to the customer data Source and "Manage users"  
![](images/Support_Support_Wiki_PowerBI_Images_Codis_BI_-_User_Access_Management_Post_Installation_1.3.2_Manage_Users.png)
3. Add User.  
![](images/Support_Support_Wiki_PowerBI_Images_Codis_BI_-_User_Access_Management_Post_Installation_1.3.3_Add_User_for_data_sources.png)
4. Then make the User Owner.  
![](images/Support_Support_Wiki_PowerBI_Images_Codis_BI_-_User_Access_Management_Post_Installation_1.3.4_Make_owner.png)
5. Now click on Share.

3. #### **Removing User Access for Customer Dashboards**
1. **Customer Workspace**
1. Go to Customer workspace and then Click on Access.   
(You can refer to steps 1\.1\.1 to 1\.1\.5\)
2. Then Click on 3 dots and Remove the user.  
![](images/Support_Support_Wiki_PowerBI_Images_Codis_BI_-_User_Access_Management_Post_Installation_2.1.2_Remove_User.png)
3. Then Close it.

3. **Customer Data Gateway**
1. Go to Settings\>Manage Gateway\>On\-premises Data Gateway.
2. Then Click on 3 dots \> Manage Users.   
(You can refer to steps 1\.2\.1 to 1\.2\.4\)
3. Then Click on Cross next to the User.  
![](images/Support_Support_Wiki_PowerBI_Images_Codis_BI_-_User_Access_Management_Post_Installation_2.2.3_Remove_User.png)
4. Then Click on Remove on pop up window and Close the Manage user window.

5. **Customer Data Sources**
1. Go to Data Sources tab.
2. Then click on 3 dots next to company data source \> Manage Users.   
(You can refer to steps 1\.3\.1 to 1\.3\.2\)
3. Then Click on Cross next to the user.  
![](images/Support_Support_Wiki_PowerBI_Images_Codis_BI_-_User_Access_Management_Post_Installation_2.3.3_Remove_user.png)
4. Then Click on Remove on pop up window and then Close the Manage user window.

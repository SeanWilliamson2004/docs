To create the Addon Package, you will require the Sage Add\-on Packager [https://ask.sage.co.uk/scripts/ask.cfg/php.exe/enduser/std\_adp.php?p\_faqid\=29725](https://ask.sage.co.uk/scripts/ask.cfg/php.exe/enduser/std_adp.php?p_faqid=29725)

Supporting Files are held in the Sage200SupportingFiles [https://dev.azure.com/codislimited/CodisDevelopment/_git/Sage200SupportingFiles](https://dev.azure.com/codislimited/CodisDevelopment/_git/Sage200SupportingFiles) repo

1. Get the MSI installer for the Excelerator S200 product you want to create the addon for.
2. Run an administrative install to extract the assemblies.

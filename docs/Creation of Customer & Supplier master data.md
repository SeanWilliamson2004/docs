# 1\. Creation of Customers

**Transaction Code: GESBPC       Description: Customers creation** 

Customers are those business partners to which you sell products and/or services. They are available for selection in such tasks as Orders in the Sales module or Customer BP Invoices in the A/P\-A/R accounting module. 

• Use the Customers task (GESBPC) on the Common data \> BPs menu to create and modify customer information. \- Only information different to what has already been discussed in the BPs and Customer Category tasks is included in this topic. 

• When creating a customer specific record, we must select a customer category and enter a customer ID. \- If we have already defined a BP with the same ID, the information defined for the BP defaults to the customer record created. \- If we are creating a customer that was not previously defined as a BP, the BP record is created when creating the customer.

• Information defined or modified for a customer in the Customers task, automatically updates the corresponding BP record. 

**Path: Common Data \-\> BPs \-\> Customers** 

# 2\. Creation of Supplier

**Transaction Code: GESBPS Description: Supplier creation** 

Suppliers are those business partners from which you purchase products and/or services. They are available for selection in such tasks as Orders in the Purchasing module or Supplier BP Invoices in the A/P\-A/R accounting module. 

• Use Suppliers (GESBPS) on the Common data \> BPs menu to create and modify supplier information. \- Only information different to what has already been discussed in the BPs and Supplier Categories tasks is included in this topic. 

• When creating a supplier specific record, you must select a supplier category and enter a supplier ID. \- If you have already defined a BP with the same ID, the information defined for the BP defaults to the supplier record created. \- If you are creating a supplier that was not previously defined as a BP, the BP record is created when creating the supplier. 

• Information defined or modified for a supplier in the Suppliers task, automatically updates the corresponding BP record. 

**Path: Common Data\-\> BPs\-\> Suppliers**

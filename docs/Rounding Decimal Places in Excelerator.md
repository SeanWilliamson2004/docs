# Solution:

There is an option to "Round value decimal places"  if you tick this Excelerator will round to 2 decimals for you. 

Sage will not accept more than 2 decimals.

![](images/Support_Support_Wiki_Images_Images_for_solutions_decimal_1.jpg)

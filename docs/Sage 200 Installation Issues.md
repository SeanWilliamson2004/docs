# Sage 200 Server Installation Issues

  


1\. Error:  "**There is a problem with Windows Installer package.A progran run as a part of the setup did not finish as expected. Contact your support personnel or package vendor.**"  


Resolution: Run Command Prompt as Administrator. Navigate to Sage 200 Installer setup folder and then run **Sage200\.msi**  


  


2\. ["**The account must be a member of the Sage200Admins**" group while selecting Sage 200 Services user during Sage 200 Server install.](https://gb-kb.sage.com/portal/app/portlets/results/viewsolution.jsp?solutionid=200427112516207)

  


3\. [Checking Active Directory configuration for Sage 200 Server Installation.](https://gb-kb.sage.com/portal/app/portlets/results/viewsolution.jsp?solutionid=200427112312285)

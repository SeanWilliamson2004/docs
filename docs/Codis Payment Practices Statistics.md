## Overview

The Codis Payment Practices Statistics (PPS) is solution to the demands of the Payment Practices and Performance requirements regulations [https://www.gov.uk/government/publications/business\-payment\-practices\-and\-performance\-reporting\-requirements](https://www.gov.uk/government/publications/business-payment-practices-and-performance-reporting-requirements).

The solution is a web\-based solution that uses the new Codis IP Solution web interface, but interfaces with the Codis IP database for most configuration information including authentication information.

The solution collates purchasing information from groups of Sage 200 or Sage 1000 companies and places this information into a warehouse and generates a report on this information.  

There is a more indepth article on the inner workings of PPS here [PPS \- How it works.aspx](PPS - How it works.md).  

## Installing the PPS Website

See the [Installation Instructions](Codis Payment Practices Statistics Installation.md "Installation Guidelines").

## Login

You are first asked to log in using your Codis IP login.  Once logged in the page is divided into 3 sections.

## Run

This section is used to generate the data used for statistics.  You have to chose a period to report on and a group of companies to report on, and create a run.  Groups of companies are maintained as described below.  

## Report for Period

In this section you select a run and generate a report for the data.  You have to choose a run that you have already made.  It does not automatically select the last run made.

![](images/General_General_Wiki_Images_PPSReport.jpg)  

To see the statistics for a report, click on the down arror by Select Report and select either the Summary or Analysis Report 

![](images/General_General_Wiki_Images_PPSSummaryDropdown.jpg)  

Then click again on the button:

![](images/General_General_Wiki_Images_SummaryAnalysisButton.jpg)  

The statistics will appear in the bottom half of the screen:

![](images/General_General_Wiki_Images_PPSStatisticSummaryReport.jpg) 

## Maintaining Groups

Groups can be maintained from the PPS website.

Your user must have administrator rights to be able to maintain groups.  If you have these rights a "Group Maintenance" button will be displayed.  Clicking on this will display a section on the screen that allows Groups to be created and maintained.

## Entering the Narratives

The PPS submissions include mandatory narrative statements about payment practices.  This can be stored by the Codis PPS solution.  The narratives are entered against groups and can be accessed from the group maintenance section via the "Narrative" button.

All Codis logos can be found [here](https://codislimited-my.sharepoint.com/personal/administrator_codis_co_uk/_layouts/15/onedrive.aspx?id=/personal/administrator_codis_co_uk/Documents/Digital%20Marketing/Logos/Codis/Codis%20logo/Legacy). When showing the logo:  

- Don't make it too big and leave plenty of white space around it
- Always resize proportionally so that the circle is not slightly egg shaped.
- Use the vector emf version in all office products, this never pixelates.

Please follow the instructions below to license Excelerator offline.

# Process:

1\.      Run Excel and access the Excelerator ribbon to click on Select Modules.  

*\[image: clip\_image002\.png not found]*![](images/Licensing_Licensing_Wiki_Images_Offline_Licencing_Picture1.png) 

2\.      A screen displaying all the modules will be displayed. Select the module(s) you have purchased and need to licence. Click Ok to proceed

![](images/Licensing_Licensing_Wiki_Images_Offline_Licencing_Picture2.png)  

3\.      Click on "Register Now"  
![](images/Licensing_Licensing_Wiki_Images_Offline_Licencing_Picture3.png)*\[image: clip\_image006\.png not found]*  

4\.      Registration Wizard will open, select Next  
*\[image: clip\_image008\.png not found]*![](images/Licensing_Licensing_Wiki_Images_Offline_Licencing_Picture4.png)  

5\.      Enter User's name and email address, and select Next  
*\[image: clip\_image010\.png not found]*![](images/Licensing_Licensing_Wiki_Images_Offline_Licencing_Picture5.png)  

6\.      Click on Register Offline  
*\[image: clip\_image012\.png not found]*![](images/Licensing_Licensing_Wiki_Images_Offline_Licencing_Picture6.png)  

7\.      Offline Registration window will open, copy the Registration code and send to licensing@codis.co.uk.

 ![](images/Licensing_Licensing_Wiki_Images_Offline_Licencing_Picture7.png)  

*\[image: clip\_image014\.png not found]*8\.      You'll receive an Activation Code, paste that in the above window and click on "Accept"  

 ![](images/Licensing_Licensing_Wiki_Images_Offline_Licencing_Picture8.png)  

9\.      Licences will be updated.

*\[image: clip\_image018\.png not found]*![](images/Licensing_Licensing_Wiki_Images_Offline_Licencing_Picture9.png)

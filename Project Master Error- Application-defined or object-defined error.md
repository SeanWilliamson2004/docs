Excelerator tab doesnot display for V1 Project Master.

Doesnot register **sesprojectmasteraddin.dll** file.

Solution: Replace **Designer** folder in C:\\Program Files (x86\)\\Common Files and register **MSADDNDR.DLL** file via command prompt successfully and register sesprojectmasteraddin.dll file again. It works.

## Recommendation

Codis recommends that Client upgrade to the Excelerator Enterprise Edition. 

  
It is the client\-server edition of Excelerator and as such one of its major features is centralised licensing. This means that all the licence details for both the software and the users are held centrally on the server instead of individual computers, allowing the user to access Excelerator from multiple locations and devices. With the current solution Client would need to purchase two licenses if the user requires access to Excelerator on their desktop and laptop, this is not the case with the Enterprise Edition. Centralised licensing also allows the licence renewal processes to be managed centrally for a more efficient service while at the same time reducing the TCO. 

  
Enhanced Security; allowing administrators to define roles for users, controlling read/write access from Company level down to Field level. A full Auditing module is now available which will allow Client to track, record and report when and by whom every journal was created.

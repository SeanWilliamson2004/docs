You need to preview the report in the program then browse to: **C:\\Users\\XXXX\\AppData\\Local\\Temp\\Sage\\Sage200\\sage\\REPORTING**. 

Within here, you will be able to obtain a hard copy of the .report or .layout files.

**Tip:** Running preview through Report Designer program will not copy the layout/report to the temp folder

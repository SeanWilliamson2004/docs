Please instruct the reseller as follows.

1\. There should be a person in the end user company who is a licence admin (where we have their email address and they are set up in crm etc). That person should log in for the reseller when the reseller is setting things up.

2\. If that is not possible (e.g. the licence admin was off sick or they want the reseller to do it all without having to get involved) then they can give the reseller the password to the licence admin's Microsoft account and that password can be changed once the reseller has finished licensing everyone.

3\. If that is not possible (e.g. they don't want to manage licences themselves and they don't want to give the reseller the licence admin Microsoft password) then the end user can create a new email address for the reseller, e.g. k3FDS@FitnessFirst.co.uk, they can set up a Microsoft account with that email address and give that to us. The reseller will then be a second licence admin for the end user.

# Upgrade Sage 200

1. Is a T\&R licence already in place? If not, add to customer sales order.
2. Receive signed sales order from customer.
3. Create new case with Codis Support
4. Send invoice for T\&R to customer. Ensure funds cleared before placing Sage order bpsales@sage.com for T\&R licence.
5. Get customer to complete the Sage customer registration form.
6. Send customer registration form to bpservices@sage.com
7. Get MTD module details from Sage (if customer is doing group submissions). bpsales@sage.com
8. Send Sage keys and MTD module details to support@codis.co.uk

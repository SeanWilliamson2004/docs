Some years ago the RSPCA decided to use a cloud based P2P system called Proactis, and to integrate it with Sage 1000\.   Although RSPCA eventually decided to change their accountancy system and the interfaces are no longer used, it is a good example of how integration between a P2P system and S1000 can be achieved.   This page is used as a repository for information for this project.

The **Silverlight Licensing System** (*silverlight licensing*) is the name given to the version of the licensing system which uses a [silverlight application](https://www.microsoft.com/silverlight/) to issue licenses for Codis software over a web browser. As of July 2015, this is the currently used system.

## Issuing telephone licences

A telephone licence, also known as **v1 licensing**, is a licence issued over the phone, where an operator tells the customer a numeric activation key such as **471 124 502 1**. There is a different procedure to follow depending on the product's licence type.

## Generating V3 (.Net) licences

V3 licencing also known as **.Net licencing**, is when the customer can send across and activation code to licensing@codis.co.uk. The licencing team will then activate the request code and send the customer back an activation code. 

This is controlled by the version of the software installed on the user site. The procedure for licensing V3 modules are as follows: 

1. Generate a Request (at the user side)
2. Email the Request to Licensing
3. Generate Activation string (at Codis office using CRM)
4. Copy the Activation string into a notepad and Email the file to the user.
5. The user pastes in the activation string. See [http://www.codis.co.uk/excelerator\-help/quick\-installation\-instructions/licensing](http://www.codis.co.uk/excelerator-help/quick-installation-instructions/licensing) for more details about what the user sees.

We recommend copying and pasting the activation code into a Notepad and attaching this to the email for the customer.

### Standard

Pre Requisites before you can generate an activation code: 

1. The Company must exist in CRM
2. A full licence requires that there is an order and for the module etc.

### Terminal/Citrix

If a customer has downloaded and using V3 modules on Terminal/Citrix you cannot activate the code without creating a dummy module in the Standard method. Once you have created a dummy module for each module request by inputting a new order under the company. You can then activate the request code, which will allow the requester to select the licence number, that is normal 4 digits. 

It is recommended entering the 4 digit code for the company to avoid confusion and further errors.

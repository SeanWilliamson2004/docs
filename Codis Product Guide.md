**This is an overview guide to the software developed by Codis for sale. It is intended for internal purposes only, although text may be selectively used for external use.** 

## Excelerator

Excelerator is a software product that provides a two\-way interface between Excel and some Sage accounting software packages. Data can be entered onto spreadsheets and saved into the accounting package. Data can also be downloaded. 

## V1 Excelerator

Codis started development of Excelerator in 1999 and sold it first seats in 2000\. The product was developed using VB6 due to its superior MS Office integration. A range of modules was produced, mostly for Sage 500 and 1000, but also a few for Sage MMS/200\. This original product was superceded by a new version written in .net, and has since been known as V1 Excelerator. V1 is characterised by (when compared with V3\) having individual installations for each module. Each module appears as a menu item in the Add\-Ins menu.

## Other Products

### DRM

It is a tool that assists with the renumbering of Sage 1000/500 posting codes, and other Sage 1000/500 master data. More information [here](https://codislimited.sharepoint.com/sites/Wiki/Sales/Sales%20Wiki/Documents/DRM/New%20DRM/DRM.pdf). The tool tends to get used to assist with consultancy products. Ongoing support of the software is not usually required after the project. 

### GL Integrity Maintenance

This is a tool used to validate balances held on Sage 1000/500 posting codes. It has tended to be needed to be used a few times on a site by a consultant. Ongoing support of the software is not usually required after the project.

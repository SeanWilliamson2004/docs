Here are some guidance notes on using Clockify if you are in the Support or Development teams:

1. Your managers expect to see a reasonably accurate representation of how you spend your time working in Clockify.
2. You should work for at least 37\.5 hours a week, plus a 1\-hour break daily, and ensure that the 1\-hour break is recorded. Therefore, Clockify should register at least 42\.5 hours a week in total.  (Except when annual leave is taken \- see 9\. below).
3. You must provide a description of how you spent your time working. While the project and task code will often suffice, if you use a generic code, you should add some text in the description.
4. Before the introduction of Clockify, you were asked to list cases you worked on at the end of each day. With Clockify, you should try to complete details of the day's work by the end of the day.
5. The Clockify Time Tracker is recommended since recording information as soon as possible is usually the easiest.  However, this does not mean that real\-time monitoring is taking place.
6. You must complete your timesheet before Monday of the following week. It is easier to track time as you go, and Clockify makes this easier than any other system I've seen. The time recorded as spent working does not have to be contiguous. However, points 2 and 6 still apply.
7. Time entered in Clockify should not overlap beyond tasks having the same start time and end time.
8. Please record lunch breaks and other significant breaks in your time\-tracker using the "Break" project code and the "Break" task code. This does not relate to any demands your role may make regarding your availability.
9. Request time off using a Clockify time off request. This can be for annual leave, paid leave agreed with your line manager, or unpaid leave. If you are sick, please request sick time off using the Clockify time off request, if you can. You must record time off or sick leave in your time tracker as 3:45 hours per half\-day off.
10. Please record public holiday time off in your time tracker. This is because it does not appear in your timesheet approval request. Please note that all team members are subject to UK public holidays as 7:30 hrs per day under the "Bank Holiday" code under "Codis General".
11. If you have agreed to work time in lieu on a weekend for time taken off during the week with your line manager, please do not request the time off but track the time. This must be done within the same timesheet week.
